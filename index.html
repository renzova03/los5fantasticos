Proyecdo del Taller de Creatividad

Hemos creado un aplicativo que te ayuda a identificar los correos de dudosa procedencia y automáticamente elimina este correo, un fantástico sistema anti Carding para tu teléfono móvil y correo electrónico.

¿Qué es el Carding?

Es un término que describe el tráfico y el uso no autorizado de tarjetas de crédito, los delincuentes acceden de forma ilegal a través de un software de manera aleatoria a una base de datos donde se encuentra la información de tarjetas de crédito o debito cuando obtienen la información realizan pagos con ellas.

El aplicativo cuenta con estos servicios.

Identificador de correos

Cuando recibas un correo la aplicación analiza este tipo de correo y si se trata de un posible robo automáticamente bloquea la dirección del correo para no recibir más de este tipo correo basura

Antivirus

Este antivirus identifica tus descargas que hagas, si tu descarga contiene algún tipo de archivo malicioso automáticamente bloqueara el permiso de descarga.

Aseguradora de tarjeta

Contamos con un seguro de tarjeta por compras fraudulentas de internet

Bloqueo de páginas webs

Al entrar a una página web donde tengas que pagar realizaremos un análisis de esta y te daremos un aviso previo sobre el pago.
